## 📖 学习路径

### 基础统计（Statistics Basics）

这部分内容帮助你理解数据的基本特征和分布规律。

1. **[标准差（Standard Deviation）](01_standard_deviation.md)** ⭐ 基础
   - 理解数据的分散程度
   - 学习方差和标准差的计算
   - 应用：评估数据的稳定性

2. **[对数函数（Logarithm）](02_logarithm.md)** ⭐ 基础
   - 理解指数和对数的关系
   - 掌握对数的运算性质
   - 应用：信息熵、交叉熵损失函数

3. **[Softmax 函数（Softmax Function）](03_softmax.md)** ⭐⭐ 进阶
   - 理解概率分布的转换
   - 掌握 Softmax 的计算方法
   - 应用：多分类问题、注意力机制

## 🎯 如何使用这些教程

### 1. 按顺序学习
建议按照编号顺序学习，因为后面的概念可能会用到前面的知识。

### 2. 动手计算
每个教程都有应用题，建议：
- 先自己尝试解答
- 再对照参考答案
- 理解每一步的推导

### 3. 理解而非记忆
不要死记公式，重点是理解：
- 这个概念是什么
- 为什么需要它
- 怎么用它解决问题

### 4. 联系实际
思考这些数学概念在 AI 和机器学习中的应用。
