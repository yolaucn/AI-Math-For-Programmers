# 程序员的数学课 📐

> 用中学生都能看懂的方式，讲清楚人工智能背后的数学原理

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🎯 项目目标

这个项目专为以下人群设计：
- 📊 想学习人工智能但数学基础薄弱的程序员
- 🤔 看到数学公式就头疼的开发者
- 💡 想重新打好数学基础的技术人员
- 🎓 希望用直观方式理解数学概念的学习者

## ✨ 特点

- **零代码**：纯数学概念讲解，不涉及编程实现
- **零依赖**：只需要一个 Markdown 阅读器
- **循序渐进**：从基础概念到实际应用
- **通俗易懂**：用中学生都能理解的语言
- **实例丰富**：每个概念都配有详细的应用题和解答

## 🚀 如何使用

1. **克隆仓库**
   ```bash
   git clone https://github.com/yolaucn/AI-Math-For-Programmers.git
   cd AI-Math-For-Programmers
   ```

2. **开始学习**
   - 按照编号顺序阅读 `tutorials/` 目录下的教程
   - 每个教程都是独立的 Markdown 文件
   - 建议使用支持数学公式的 Markdown 阅读器（如 Typora、VS Code + Markdown Preview）

3. **动手练习**
   - 每个教程都包含应用题
   - 先自己尝试解答，再查看参考答案
   - 理解每一步的推导过程

## 📖 教程结构

每个教程都遵循统一的结构：

```
1. 符号速查表 - 快速查阅数学符号
2. 数学定义 - 严谨的数学表述
3. 算法描述 - 分步骤的计算方法
4. 应用题 - 实际问题练习
5. 参考答案 - 详细的求解过程
6. 现象解释 - 结果的实际意义
```

## 🤝 贡献指南

欢迎贡献新的教程或改进现有内容！

### 贡献方式

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/new-tutorial`)
3. 按照[教程模板](TUTORIAL_TEMPLATE.md)编写内容
4. 提交你的修改 (`git commit -m '添加：线性回归教程'`)
5. 推送到分支 (`git push origin feature/new-tutorial`)
6. 创建 Pull Request

### 内容要求

- ✅ 使用通俗易懂的语言
- ✅ 提供完整的计算步骤
- ✅ 包含实际应用场景
- ✅ 配有详细的参考答案
- ❌ 避免过于学术化的表述
- ❌ 不要假设读者有高深的数学背景

详见 [CONTRIBUTING.md](CONTRIBUTING.md)
